Este proyecto fue creado en React. Se trata de un sitio responsivo que consta de una página principal (HomePage), una para ver los detalles de cada producto (DetailPage) en donde se puede elegir cantidad y hacer la compra, una correspondiente al carrito de compras (CartPage), y un formulario de contacto (ContactPage). 
El sitio fue diseñado a partir del modelo compartido en Figma por el docente.
Se utilizó JavaScript para programar el carrito (utilizando Context), el formulario de contacto y filtrador para búsqueda por nombres de productos en el input del HomePage. 
Para definir las rutas de navegación, se utilizó la librería de React Router Dom. 
El proyecto está deployado en Vercel (https://trabajo-final-nu.vercel.app/) y con su correspondiente repositorio en GitHub (https://github.com/fedegal1984/TrabajoFinal).
